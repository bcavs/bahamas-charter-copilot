# Demo Pack — Ready-to-Send Outreach Assets

Per-captain, copy-paste WhatsApp assets for the first cold touch. Built per the
approach in [`docs/07-cold-outreach-playbook.md`](../07-cold-outreach-playbook.md):
WhatsApp-first, show-don't-ask, one personalized mini-demo each.

**How to use each entry:**
1. Send **Opener**.
2. When they reply (or after a short beat), send the **Demo** — paste the mock
   inquiry line, then the drafted reply. (Once the product is set up with their
   real trips, send an actual screenshot instead.)
3. Send **Ask**.
4. Log the touch in `captain-outreach-list.csv` (`Status` → `Demo Sent`, set
   `Last Contact Date` / `Next Follow-up`).

**Before sending, slot in the real rate** where you see `[your rate]`. The
copilot never invents a final price — keep it that way; quoting a fake number to
a captain who knows their own pricing kills the credibility.
